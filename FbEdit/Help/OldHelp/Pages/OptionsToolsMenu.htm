Tools Menu



 

● Menu item.

    The text shown on the menu.

● Command.

    The tool file (.exe, .bat).
    If the command is a single $ then an explorer with current path will be shown.
    If path is $A the application path is used.
    If path is $C the compiler path is used.
    A $ as parameter will use the current open file as parameter.
    Example: notepad.exe $
    This will open the current file in notepad.
