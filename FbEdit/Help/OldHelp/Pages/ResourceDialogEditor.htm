Dialog Editor



 

● Adding a control to the dialog.

Select a control from the toolbox to the left. Then left click on the dialog where you want your control. You can also hold the mouse button and draw it to the size you want.
NOTE!
If you hold the Ctrl key while selecting from the toolbox, the selected button will stay down. This can save time when you want several controls of the same type.

● Selecting controls.

Select a control by left clicking on the control. If you hold the Ctrl key you can select more than one control. You can also hold Shift and left click on the dialog. Hold the mouse button and draw a rectangle. All controls in the rectangle will be selected when you release the button.
You can also use Tab or Arrow keys to select a control.

● Sizeing controls.

Select a control and move the mouse to one of the six sizing rectangles. Left click and drag to the size you want.
You can also hold the Shift key and use Arrow keys to size the control. This also works on a multiselection.

● Moving controls.

Select a control by left clicking on the control. Hold the button and drag the control to where you want.
You can also hold the Ctrl key and use Arrow keys to move the control. This also works on a multiselection.



Control properties

Here you can change the looks and behaviour of your control.
Each control type has different properties.

When the properties list has the focus you can use the keyboard to navigate.
Arrow keys moves the selection up or down.
Typing the first letters of a property selects the property item.
Enter enters the edit box or selects the next item in the list of possible values.
